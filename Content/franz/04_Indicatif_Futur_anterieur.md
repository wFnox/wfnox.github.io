# 1. Name der Zeitform

| Französisch | Deutsch |
|-------------|---------|
| Futur antérieur | Futur II / vollendete Zukunft |

---

# 2. Verwendung

- Eine Handlung, die in der Zukunft **abgeschlossen sein wird**, bevor eine andere beginnt
- Wird oft in **Nebensätzen mit quand, lorsque, dès que, aussitôt que** verwendet
- Kann auch eine **Vermutung über die Vergangenheit** ausdrücken (wahrscheinlich ist etwas passiert)

> Das Futur antérieur verhält sich zur Zukunft wie das Plus-que-parfait zur Vergangenheit.

---

# 3. Bildung

> **avoir** ODER **être** (im Futur simple) + **Participe passé**

### Mit avoir:
| Person | Form |
|--------|------|
| j' | aurai parlé |
| tu | auras parlé |
| il/elle | aura parlé |
| nous | aurons parlé |
| vous | aurez parlé |
| ils/elles | auront parlé |

### Mit être:
| Person | Form |
|--------|------|
| je | serai allé(e) |
| tu | seras allé(e) |
| il/elle | sera allé(e) |
| nous | serons allés |
| vous | serez allés |
| ils/elles | seront allés |

> Gleiche être-Verben wie beim Passé composé. Das Participe passé stimmt mit dem Subjekt überein.

---

# 4. Futur simple von avoir und être (Hilfsverben)

| Person | avoir | être |
|--------|-------|------|
| je | aurai | serai |
| tu | auras | seras |
| il/elle | aura | sera |
| nous | aurons | serons |
| vous | aurez | serez |
| ils/elles | auront | seront |

### Wichtige unregelmäßige Futur-simple-Stämme (für avoir/être)

| Infinitiv | Futur-Stamm | je-Form |
|-----------|-------------|---------|
| avoir | aur- | j'aurai |
| être | ser- | je serai |
| aller | ir- | j'irai |
| faire | fer- | je ferai |
| pouvoir | pourr- | je pourrai |
| vouloir | voudr- | je voudrai |
| venir | viendr- | je viendrai |
| voir | verr- | je verrai |
| savoir | saur- | je saurai |

> 💡 Das Participe passé beim Futur antérieur folgt denselben Regeln wie beim Passé composé – inklusive aller unregelmäßigen Partizipien!

- **Quand tu arriveras, j'aurai déjà mangé.** → Wenn du ankommst, werde ich schon gegessen haben.
- **Dès qu'il aura fini ses devoirs, il pourra sortir.** → Sobald er seine Hausaufgaben beendet hat, darf er rausgehen.
- **Elle aura oublié notre rendez-vous.** → Sie wird unseren Termin wohl vergessen haben. *(Vermutung)*
- **Dans deux heures, nous serons partis.** → In zwei Stunden werden wir abgereist sein.

# 6. Signalwörter

| Signalwort | Bedeutung |
|------------|-----------|
| quand | wenn (zukünftig) |
| lorsque | wenn, als |
| dès que | sobald |
| aussitôt que | sobald |
| une fois que | sobald (einmal) |
| avant que | bevor |
| dans … (heures/jours) | in … (Stunden/Tagen) |
