# 1. Name der Zeitform

| Französisch | Deutsch |
|-------------|---------|
| Passé composé | Perfekt / vollendete Vergangenheit |

---

# 2. Verwendung

- **Abgeschlossene Handlungen** in der Vergangenheit (mit klarem Endpunkt)
- Handlungen, die **einmalig** oder **zu einem bestimmten Zeitpunkt** stattfanden
- **Aufeinanderfolgende Ereignisse** in der Vergangenheit
- Handlungen, die eine **bestimmte Anzahl von Malen** vorkamen

> Das Passé composé wird im Gegensatz zum Imparfait benutzt, wenn eine Handlung abgeschlossen ist.

---

# 3. Bildung

> **avoir** ODER **être** (im Présent) + **Participe passé**

### Hilfsverb: avoir
Die meisten Verben bilden das Passé composé mit **avoir**.

| Person | Form |
|--------|------|
| j' | ai parlé |
| tu | as parlé |
| il/elle | a parlé |
| nous | avons parlé |
| vous | avez parlé |
| ils/elles | ont parlé |

### Hilfsverb: être
Bewegungsverben und reflexive Verben nehmen **être**.  
Das Participe passé stimmt dann mit dem **Subjekt** überein!

| Person | Form |
|--------|------|
| je (m) | suis allé |
| je (f) | suis allée |
| tu (m) | es allé |
| il | est allé |
| elle | est allée |
| nous | sommes allés |
| vous | êtes allés |
| ils | sont allés |
| elles | sont allées |

### Wichtige être-Verben (DR & MRS VANDERTRAMP)
aller, venir, partir, arriver, entrer, sortir, monter, descendre, naître, mourir, rester, tomber, retourner, passer (par), devenir, revenir + alle **réfléchis**

---

# 4. Bildung des Participe passé

| Infinitiv-Endung | Participe passé | Beispiel |
|------------------|-----------------|---------|
| -er | **-é** | parler → parlé |
| -ir | **-i** | finir → fini |
| -re | **-u** | vendre → vendu |

# 5. Beispielsätze

- **J'ai mangé une pomme.** → Ich habe einen Apfel gegessen.
- **Elle est partie à huit heures.** → Sie ist um acht Uhr abgefahren.
- **Nous avons visité Paris l'année dernière.** → Wir haben letztes Jahr Paris besucht.
- **Il s'est levé tôt ce matin.** → Er ist heute Morgen früh aufgestanden.
