# 2. Verwendung

- Dinge, die **gerade jetzt** passieren
- **Allgemeine Wahrheiten** und Fakten
- **Regelmäßige / gewohnheitsmäßige** Handlungen
- Handlungen, die in **naher Zukunft** geplant sind
- Zustände, die **andauern**

---

# 3. Bildung

> **Verbstamm + Endung**

Der Stamm wird gebildet, indem man die Infinitivendung (-er, -ir, -re) entfernt.

**Beispiel mit „parler" (sprechen):**

| Person | Endung | Form |
|--------|--------|------|
| je | -e | je parle |
| tu | -es | tu parles |
| il/elle | -e | il parle |
| nous | -ons | nous parlons |
| vous | -ez | vous parlez |
| ils/elles | -ent | ils parlent |

---

# 4. Verb-Endungen

### -ER Verben (parler)
| Person | Endung |
|--------|--------|
| je | **-e** |
| tu | **-es** |
| il/elle | **-e** |
| nous | **-ons** |
| vous | **-ez** |
| ils/elles | **-ent** |

### -IR Verben (finir)
| Person | Endung |
|--------|--------|
| je | **-is** |
| tu | **-is** |
| il/elle | **-it** |
| nous | **-issons** |
| vous | **-issez** |
| ils/elles | **-issent** |

### -RE Verben (vendre)
| Person | Endung |
|--------|--------|
| je | **-s** |
| tu | **-s** |
| il/elle | **–** (keine Endung) |
| nous | **-ons** |
| vous | **-ez** |
| ils/elles | **-ent** |

---

# 5. Beispielsätze

- **Je mange une pomme.** → Ich esse einen Apfel.
- **Il fait beau aujourd'hui.** → Es ist heute schön (Wetter).
- **Nous habitons à Paris.** → Wir wohnen in Paris.
- **Le soleil se lève à l'est.** → Die Sonne geht im Osten auf. *(allgemeine Wahrheit)*

### Rechtschreibliche Ausnahmen bei -ER Verben

| Typ | Regel | Beispiel |
|-----|-------|---------|
| -ger Verben | nous + **e** vor -ons | nous mang**e**ons |
| -cer Verben | nous: c → **ç** | nous commen**ç**ons |
| e + Konsonant + er | e → è (außer nous/vous) | j'achète, il lève |
| é + Konsonant + er | é → è (außer nous/vous) | il préfère, je répète |
| -yer Verben | y → i (außer nous/vous) | j'envoie, ils nettoient |
| -eler / -eter | Doppelkonsonant (außer nous/vous) | j'appelle, je jette |
