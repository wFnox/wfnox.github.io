# 1. Name der Zeitform

| Französisch | Deutsch |
|-------------|---------|
| Passé antérieur | Vorzeitige Vergangenheit (literarisch) |

---

# 2. Verwendung

- Eine Handlung, die **unmittelbar vor** einer anderen Vergangenheitshandlung abgeschlossen war
- Wird **fast ausschließlich in der Literatur** und in formellen Texten verwendet
- Ersetzt das **Plus-que-parfait** in Nebensätzen nach **quand, lorsque, dès que, aussitôt que, à peine … que**
- Im modernen Französisch (gesprochen) ersetzt durch das **Plus-que-parfait**

> ⚠️ Diese Zeitform kommt fast nur in **literarischen Texten** vor – im Alltag sehr selten!

---

# 3. Bildung

> **avoir** ODER **être** (im Passé simple) + **Participe passé**

### Mit avoir:
| Person | Form |
|--------|------|
| j' | eus parlé |
| tu | eus parlé |
| il/elle | eut parlé |
| nous | eûmes parlé |
| vous | eûtes parlé |
| ils/elles | eurent parlé |

### Mit être:
| Person | Form |
|--------|------|
| je | fus allé(e) |
| tu | fus allé(e) |
| il/elle | fut allé(e) |
| nous | fûmes allés |
| vous | fûtes allés |
| ils/elles | furent allés |

---

# 4. Passé simple von avoir und être (Hilfsverben)

| Person | avoir | être |
|--------|-------|------|
| je | eus | fus |
| tu | eus | fus |
| il/elle | eut | fut |
| nous | eûmes | fûmes |
| vous | eûtes | fûtes |
| ils/elles | eurent | furent |

### Weitere unregelmäßige Passé-simple-Formen

| Infinitiv | je | il/elle | ils/elles |
|-----------|----|---------|-----------|
| faire | fis | fit | firent |
| prendre | pris | prit | prirent |
| mettre | mis | mit | mirent |
| voir | vis | vit | virent |
| venir | vins | vint | vinrent |
| dire | dis | dit | dirent |
| écrire | écrivis | écrivit | écrivirent |
| naître | naquis | naquit | naquirent |
| mourir | mourus | mourut | moururent |

> 💡 Das Participe passé beim Passé antérieur ist identisch mit dem beim Passé composé.

---

# 5. Beispielsätze

- **Dès qu'il eut fini son discours, les applaudissements éclatèrent.** → Sobald er seine Rede beendet hatte, brach der Applaus los.
- **Aussitôt qu'elle fut arrivée, la cérémonie commença.** → Sobald sie angekommen war, begann die Zeremonie.
- **À peine eut-il parlé qu'on le reconnut.** → Kaum hatte er gesprochen, erkannte man ihn.

---

# 6. Signalwörter

| Signalwort | Bedeutung |
|------------|-----------|
| dès que | sobald |
| aussitôt que | sobald |
| lorsque / quand | als |
| à peine … que | kaum … als |
| une fois que | sobald (einmal) |
| après que | nachdem |

> 💡 Das Passé antérieur erscheint **immer im Nebensatz**, während der Hauptsatz im **Passé simple** steht.
