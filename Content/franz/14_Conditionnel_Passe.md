# 1. Name der Zeitform

| Französisch | Deutsch |
|-------------|---------|
| Conditionnel passé | Konjunktiv II Perfekt / vollendeter Konditional |

---

# 2. Verwendung

- **Vergangene Hypothesen** – was hätte passieren können, aber nicht passiert ist
- In **Bedingungssätzen** mit „si + Plus-que-parfait" (Typ III)
- **Vorwürfe** oder Bedauern über vergangene Ereignisse
- **Nicht erfüllte Wünsche** in der Vergangenheit
- **Nicht bestätigte Aussagen** in der Vergangenheit (journalistisch)

> **Vergleich der si-Satz-Typen:**  
> Typ II: Si j'avais de l'argent, **j'achèterais** une voiture. *(Conditionnel présent)*  
> Typ III: Si j'avais eu de l'argent, **j'aurais acheté** une voiture. *(Conditionnel passé)*

---

# 3. Bildung

> **avoir** ODER **être** (im Conditionnel présent) + **Participe passé**

### Mit avoir:
| Person | Form |
|--------|------|
| j' | aurais parlé |
| tu | aurais parlé |
| il/elle | aurait parlé |
| nous | aurions parlé |
| vous | auriez parlé |
| ils/elles | auraient parlé |

### Mit être:
| Person | Form |
|--------|------|
| je | serais allé(e) |
| tu | serais allé(e) |
| il/elle | serait allé(e) |
| nous | serions allés |
| vous | seriez allés |
| ils/elles | seraient allés |

> Gleiche être-Verben wie beim Passé composé. Das Participe passé stimmt mit dem Subjekt überein.

---

# 4. Conditionnel présent von avoir und être (Hilfsverben)

| Person | avoir | être |
|--------|-------|------|
| je | aurais | serais |
| tu | aurais | serais |
| il/elle | aurait | serait |
| nous | aurions | serions |
| vous | auriez | seriez |
| ils/elles | auraient | seraient |

### Accord du participe passé (Angleichung)

| Hilfsverb | Angleichung? | Beispiel |
|-----------|-------------|---------|
| avoir | ❌ Nein | j'aurais mangé |
| être | ✅ Ja, mit dem Subjekt | elle serait allée → **allée** |

> 💡 Das Participe passé folgt denselben Regeln wie im Passé composé – inklusive aller unregelmäßigen Formen. Die Hilfsverben (avoir/être) im Conditionnel présent verwenden die unregelmäßigen Stämme **aur-** und **ser-**.

---

# 5. Beispielsätze

- **Si j'avais étudié, j'aurais réussi l'examen.** → Wenn ich gelernt hätte, hätte ich die Prüfung bestanden.
- **Tu aurais dû me le dire !** → Du hättest mir das sagen sollen! *(Vorwurf)*
- **J'aurais voulu voyager plus.** → Ich hätte mehr reisen wollen. *(nicht erfüllter Wunsch)*
- **Le président aurait démissionné.** → Der Präsident soll zurückgetreten sein. *(journalistisch, unbestätigt)*

# 6. Signalwörter

| Signalwort | Bedeutung |
|------------|-----------|
| si + Plus-que-parfait | wenn … gehabt hätte |
| à ta place | an deiner Stelle |
| j'aurais dû | ich hätte sollen |
| tu aurais pu | du hättest können |
| on aurait pu | man hätte können |
| il aurait fallu | es wäre nötig gewesen |
| peut-être | vielleicht |
| probablement | wahrscheinlich |
