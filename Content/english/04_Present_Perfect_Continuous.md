# 1. Name der Zeitform

| Englisch | Deutsch |
|----------|---------|
| Present Perfect Continuous | Verlaufsform des Perfekts |

---

# 2. Verwendung

- Eine Handlung, die **in der Vergangenheit begann** und **bis jetzt andauert** (Betonung auf Dauer)
- Eine **kürzlich beendete** Handlung, deren **Auswirkungen noch sichtbar** sind
- Zur Erklärung eines **Zustands oder Ergebnisses** in der Gegenwart

> **Unterschied zu Present Perfect Simple:**
> - *I have read this book.* → Ich habe das Buch (fertig) gelesen. *(Ergebnis zählt)*
> - *I have been reading this book.* → Ich lese dieses Buch schon eine Weile. *(Dauer zählt)*

---

# 3. Bildung

> **have / has + been + Verb-ing**

| Person | Form (work) |
|--------|-------------|
| I | **have been** working |
| you | **have been** working |
| he / she / it | **has been** working |
| we | **have been** working |
| you | **have been** working |
| they | **have been** working |

> ⚠️ Gleiche -ing Regeln wie beim Present Continuous beachten!

### Verneinung
> **have / has + not + been + Verb-ing**

- *I have not (haven't) been sleeping well.*
- *She has not (hasn't) been feeling well lately.*

### Frage
> **Have / Has + Subjekt + been + Verb-ing?**

- *How long have you been waiting?*
- *Has it been raining all day?*

---

# 4. Beispielsätze

- **I have been studying for three hours.** → Ich lerne schon seit drei Stunden.
- **She has been working here since January.** → Sie arbeitet hier seit Januar.
- **You look tired. Have you been running?** → Du siehst müde aus. Bist du gerannt?
- **It has been raining all morning.** → Es regnet den ganzen Morgen.

---

# 5. Signalwörter

| Signalwort | Bedeutung |

| since | seit (Zeitpunkt) |

| all day / morning | den ganzen Tag / Morgen |
| lately | in letzter Zeit |
| recently | kürzlich |
